🧪 Dissolved Gas Analyzer (DGA) Web Tool

A modern web-based Dissolved Gas Analysis (DGA) tool designed to help analyze transformer oil health using key gas concentration data.

This project currently includes a fully functional frontend (HTML, CSS, JavaScript) and aims to evolve into a powerful full-stack analytical platform with Python-powered diagnostics and advanced interpretation models.

🚀 Project Vision

The goal of this project is to build an intelligent DGA system that can:

Analyze transformer oil gas data
Detect faults using standard DGA methods
Provide actionable insights for maintenance teams
Eventually integrate AI/ML models for predictive failure detection
